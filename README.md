# 随心记

轻量图文社区：把日常写成可分享的动态，在广场发现同好。

## 在线体验

**[https://get.fs-hck.top/notes/](https://get.fs-hck.top/notes/)**

## 亮点

- **块式编辑**：多段文字、图文混排、相册网格等布局，一条动态自由组合。
- **心情与天气**：记录当下情绪，新笔记可自动带上当前天气（Open-Meteo）。
- **广场与发现**：公开笔记流、搜索、热门推荐；未登录也可浏览精选内容。
- **个人空间**：草稿与已发布分栏、相册聚合、分享记录；可逛他人主页。
- **一键分享**：生成分享图/链接，适配微博等常见渠道（具体以站内为准）。
- **自建友好**：Docker 一键部署，PostgreSQL 持久化，支持子路径与 HTTPS 配置。

## 风格

Vue 3 + Tailwind，浅天蓝主色、毛玻璃顶栏与卡片式信息流；标题与正文可选用霞骛文楷、思源宋体等网页字体，偏阅读向、不花哨。

## 功能一览

| 模块 | 说明 |
|------|------|
| 首页 / 随心广场 | 公开笔记、搜索、热门、发布入口 |
| 动态 | 关注时间线式浏览（需登录） |
| 我的 | 笔记、草稿、相册、分享历史、个人资料与背景图 |
| 编辑器 | 块编辑、心情、天气、保存草稿 / 发布 |
| 笔记详情 | 阅读、分享 |
| 用户空间 | 按用户浏览公开内容 |
| 登录注册 | 账号体系（可选短信校验，见部署配置） |

## 技术栈（简）

前端：Vue 3、Vite、Pinia、Vue Router、Axios。  
后端：FastAPI、Uvicorn、PostgreSQL。  
部署：Docker Compose、Nginx 静态站 + API 反代。

详细部署见 [`deploy/DEPLOY.md`](deploy/DEPLOY.md)。
